# 02. スコープ

## 2.1 ゴール
1. 16 進/ASCII の同期ビューでの閲覧・編集
2. オフセット指定による高速ナビゲーションとジャンプ
3. シンプルな検索・置換、差分確認、ブックマーク
4. バイナリファイルの安全な保存（バックアップ・ロールバック）
5. Rust crate としてライブラリ API を公開し、将来的な GUI/TUI 拡張に耐える構造

## 2.2 非ゴール
- 1 GB を超える巨大ファイルの完全編集（読み込みは可能だが、ストリーミング編集は次期課題）
- 逆アセンブルやプロトコル解析などヘキサビュー以外の高度可視化
- GUI（SDL/egui など）の実装
